# Snowcapped

## Dependencies

- yarn
- Docker

## Set project

```make
make setup
```

## Build project

```make
make build
```

## Install dependencies

```make
make install
```

## Set database

```make
make init-db
```

## Run migrations

```make
make db-migrate
```

## Revert last migration

```make
make db-migrate-revert
```

## Run develop mode

```make
make dev
```

## Run debug mode

```make
make debug
```

## Clean docker images

```make
make down
```
